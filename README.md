# Smart Language Switcher for Windows (AutoHotkey v2)

Легкий и надежный скрипт на **AutoHotkey v2**, который меняет раскладку клавиатуры (RU/EN) по нажатию комбинации `Ctrl + Shift`. 

В отличие от стандартного переключателя Windows, этот скрипт работает без задержек и стабильно переключает язык в **любых** окнах, включая системные утилиты.

## ✨ Особенности / Преимущества

* **Работает везде:** Благодаря умному определению фокуса (`ControlGetFocus`), язык успешно переключается даже в окне «Выполнить» (`Win + R`), Диспетчере задач и консолях.
* **Любой порядок нажатия:** Неважно, нажали вы сначала `Ctrl` или `Shift` — раскладка переключится.
* **Защита от дребезга (Double-strike protection):** Исключает случайное двойное переключение языка при одновременном зажатии клавиш.
* **Автоматический запуск от администратора:** Скрипт сам запрашивает повышенные права при старте, что необходимо для взаимодействия с системными окнами.

## 🚀 Требования и запуск

1. Установите [AutoHotkey v2](https://www.autohotkey.com/).
2. Скачайте файл `hotkey.ahk` (или скопируйте его код) и запустите его.

### ⚠️ Важный шаг: отключите стандартное переключение в Windows
Чтобы Windows не перехватывала нажатие `Ctrl + Shift` и не мешала работе скрипта, отключите стандартное сочетание клавиш в системе:
1. Откройте **Параметры** (`Win + I`) -> **Время и язык** -> **Ввод** -> **Дополнительные параметры клавиатуры**.
2. Нажмите **Сочетание клавиш для языков ввода**.
3. Выберите строку *«Переключить язык ввода»*, нажмите кнопку **Сменить сочетание клавиш...**
4. Установите значение **«Не назначено»** для смены языка ввода (или выберите `Alt + Shift`, если планируете использовать `Ctrl + Shift` только через скрипт). Нажмите *ОК* и *Применить*.

> 💡 **Совет:** Чтобы скрипт работал всегда, добавьте его ярлык в папку автозагрузки Windows (`Win + R` -> введите `shell:startup` -> перетащите туда ярлык скрипта).

## 🛠️ Технические детали (для разработчиков)

Смена раскладки реализована через прямой вызов Windows API (`PostMessage` и `GetKeyboardLayout`). Скрипт не симулирует нажатие клавиш, а отправляет системное сообщение `WM_INPUTLANGCHANGEREQUEST (0x0050)` конкретно тому элементу управления, который сейчас находится в фокусе. Это гарантирует 100% срабатывание без "залипания" клавиш.
